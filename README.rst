Installation
------------

zeroshot_topics is distributed on `PyPI <https://pypi.org>`_ as a universal
wheel and is available on Linux/macOS and Windows and supports
Python 3.7+ and PyPy.

.. code-block:: bash

    $ pip install zeroshot_topics
